* WordCounter

  统计文件的行数，所有单词个数，同时还会记录每个单词首次出现的行数和总共出现的次数。
  [[file:demo.png][demo]]

** 编译

   #+BEGIN_SRC shell :session :results output
     mkdir build
     cmake -S . -B ./build 
     make -C ./build
     ./build/WordCounter
   #+END_SRC

** 使用

   + 测试文件 `test.txt` 有如下内容：

   #+BEGIN_SRC shell :session :results output
     This is a word counter.
     For test:

     one three-three four-
     four four
     Five FIve FIVe FIVE five.
   #+END_SRC

   + 统计结果

   #+BEGIN_SRC shell :session :results output
